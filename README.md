# ABOUT MY DOG

愛犬チャコのプロフィールサイトです。  
Canvaで作成したデザインをもとに、HTML/CSSでWebサイトとして制作しました。

## 使用技術

- HTML
- CSS
- Flexbox
- レスポンシブデザイン
- GitHub

## 工夫したところ

- カード型レイアウトで情報を整理
- タブレット・スマホ表示に対応
- 円グラフ風の性格・特徴欄をCSSで作成
- ページ内リンクと上に戻るボタンを実装
- やわらかい配色と丸みのあるデザインに統一

## 制作目的

HTML/CSSの復習と、Canvaデザインを実際のWebサイトに変換する練習として制作しました。
